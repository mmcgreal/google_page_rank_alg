Home

Algorithm used by Google to rank the pages across the WWW by popularity.
- Called a link analysis algorithm

The rank value of a page gives the overall popularity of that page
- Determined by how many times that page is referenced, as a hyperlink, in other webpages

The algorithm is defined recursively
- A page with a high page rank that hyperlinks another page will cause that page’s rank to go up by a higher amount

History

PageRank was developed by Larry Page and Sergey Brin in 1996 at Stanford University
- Came from a research project to develop a new search engine

After the paper on the project was published in 1998, Page and Brin founded Google Inc

The PageRank process is patented, but to the University, not Google Inc. The University received 1.8 million shares in exchange for Google using the patent, and sold the shares in 2005 for $336 million.

Algorithm Description

Google’s PageRank algorithm outputs a probability distribution used to represent the likelihood that a person randomly clicking on links will arrive at any particular page

Can be calculated for any size of documents

Computations undergo multiple iterations (“passes”) in order to adjust the approximate PageRank values closer to the theoretical true ones

Probability is a value between 0 and 1. A PageRank of 0.5 means there’s a 50% chance that a person clicking on a random link will be directed to the document.

Calculated recursively by the PageRanks of pages that link to this page

Determined by how many other pages reference this one and by how many outbound links the page contains

Done by individual pages, not entire websites

Algorithm

PR(A) = (1-d) + d(PR(T1)/C(T1) + … + PR(Tn)/C(Tn))

PR(A) is the PageRank of Page A
d is a dampening factor between 0 and 1
PR(Ti) is the PageRank value of pages Ti which link to Page A
C(Ti) is the number of outbound links on Page Ti

Notes:
Multiple outbound links to a single page are ignored; this is only counted as one outbound link
The more outbound links on Page Ti decrease its influence on Page A’s PageRank
The dampening factor is the probability that a person will stop clicking (it reduces the influence of outbound links to Page A).
The dampening factor is sually set at 0.85. The more outbound links on Page Ti decrease its influence on Page A’s PageRank.


Variations

SERP (Search Engine Results Page)
- The actual result (the links) given when prompted by a search query
- Uses the PageRank algorithm, along with 200 more factors to determine a page’s final value
- Differs from PageRank because it measures both “authority and popularity,” aka relevance and reputation. PageRank is simply a measure of a page’s reputation

The Intentional Surfer Model
- Reflects the importance of a particular page as a function of how many actual visits it receives by real users (PageRank is considered random surfer model)
